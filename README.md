# My Custom ERP System

A Laravel-based business management system with modules for HR, payroll, attendance, and operations.

## Features
- Employee management
- Attendance monitoring
- Payroll processing
- Role-based access control
- Dashboard and reports

## Tech Stack
- Laravel
- PHP
- MySQL
- Bootstrap / Vite

## Installation
1. Clone the repository
2. Run `composer install`
3. Run `npm install`
4. Copy `.env.example` to `.env`
5. Run `php artisan key:generate`
6. Configure database
7. Run `php artisan migrate --seed`
8. Run `npm run dev`
